## 🔨핵심기능
* `맛별 검색` 맛 별 막걸리 검색 기능
* `평가 작성` 막걸리의 맛과 특징을 평가하고 다른 사람들과 공유
* `라벨 검색` 라벨을 통해 원하는 제품을 빠르게 찾아볼 수 있는 라벨 검색 기능
